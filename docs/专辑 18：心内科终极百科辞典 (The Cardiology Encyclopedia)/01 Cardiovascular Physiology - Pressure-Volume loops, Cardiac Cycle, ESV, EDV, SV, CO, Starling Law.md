::: tip ⚡️ 核心考点 (30s速读)
*   **核心考点**：心脏是一个“工厂”，其核心功能指标是**心输出量**，由**心率**和**每搏输出量**决定。理解**压力-容积环**是掌握心室功能、**前负荷**、**后负荷**和**心肌收缩力**的关键。
*   **临床意义**：通过**舒张末期容积**、**收缩末期容积**和**射血分数**评估心脏泵血功能。**Starling定律**解释了心脏如何根据回心血量自动调节其泵血量，是理解心衰代偿机制的基础。
:::

## 🧠 深度精讲

*   **心脏的“工厂”模型**：心脏被比喻为一个工厂。**静脉**是原料输入口（**前负荷**），**动脉**是成品输出口。工厂的总产量（**心输出量**）等于工人数量（**心率**）乘以每个工人的生产效率（**每搏输出量**）。
*   **心输出量**：指一侧心室每分钟泵出的血液总量，**CO = 心率 × 每搏输出量**。正常成人静息时约为5 L/min。左右心室的心输出量相等。
*   **每搏输出量的决定因素**：每次心跳射出的血量取决于三个核心因素：
    1.  **前负荷**：心脏收缩前（舒张末期）心肌纤维被拉长的程度，主要由**舒张末期容积**决定。这对应Starling定律。
    2.  **后负荷**：心室射血时需要克服的阻力，主要指**主动脉压**。
    3.  **心肌收缩力**：心肌细胞本身收缩的强度和速度，与心脏的“劲”有关。
*   **心动周期中的关键容积**：
    *   **舒张末期容积**：心室在充盈结束后、收缩开始前的血容量，代表**前负荷**。
    *   **收缩末期容积**：心室射血结束后、舒张开始前剩余的血容量。
    *   **每搏输出量 = 舒张末期容积 - 收缩末期容积**。
    *   **射血分数**：**每搏输出量 / 舒张末期容积**，是评估心室泵血效率的重要指标（正常>55%）。
*   **Starling定律**：在一定范围内，心脏舒张末期容积（前负荷）越大，心肌纤维初长度越长，收缩力就越强，从而使每搏输出量增加。这是心脏固有的、重要的自我调节机制。
*   **压力-容积环**：这是一个描述单个心动周期中，心室压力与容积变化关系的图形。它是理解上述所有概念的“金钥匙”：
    *   **环的面积**代表**每搏功**。
    *   **环的宽度**代表**每搏输出量**。
    *   **右侧边界**的横坐标位置代表**舒张末期容积**。
    *   **左侧边界**的横坐标位置代表**收缩末期容积**。
    *   **环的顶部高度**反映**后负荷**。
    *   环的整体位置和斜率变化反映**心肌收缩力**的变化。

## 📚 双语术语表 (Terminology)
| 英文术语 | 中文翻译 | 定义/解释 |
| :--- | :--- | :--- |
| Cardiac Output | 心输出量 | 一侧心室每分钟泵出的血液总量，CO = HR × SV。 |
| Heart Rate | 心率 | 心脏每分钟搏动的次数。 |
| Stroke Volume | 每搏输出量 | 一次心搏中，一侧心室射出的血液量，SV = EDV - ESV。 |
| End-Diastolic Volume | 舒张末期容积 | 心室舒张末期（收缩前）的血液容积，反映前负荷。 |
| End-Systolic Volume | 收缩末期容积 | 心室收缩末期（射血后）的血液容积。 |
| Ejection Fraction | 射血分数 | 每搏输出量占舒张末期容积的百分比，EF = (SV / EDV) × 100%。 |
| Starling‘s Law | 斯塔林定律 | 心肌收缩力随心肌纤维初长度（前负荷）增加而增强的规律。 |
| Preload | 前负荷 | 心脏收缩前所承受的负荷，通常指舒张末期容积或压力。 |
| Afterload | 后负荷 | 心室射血时需要克服的阻力，主要指主动脉压。 |
| Contractility | 心肌收缩力 | 心肌细胞在不依赖前、后负荷的情况下，其内在的收缩能力。 |
| Pressure-Volume Loop | 压力-容积环 | 描述一个心动周期中心室压力与容积变化关系的闭合曲线图。 |
| Cardiac Cycle | 心动周期 | 心脏一次收缩和舒张的机械活动全过程。 |
