::: tip ⚡️ 核心考点 (30s速读)
*   **核心考点**：心脏在胸腔内的真实解剖方位是心尖朝前、向下、向左，心底朝后。传统图示中的“情人节”位（心尖垂直向下）是解剖室检查体位，并非其在体内的自然位置。
*   **临床意义**：准确理解心脏的解剖方位是进行心脏听诊、影像学检查（如超声心动图、CT）以及心导管介入等临床操作的基础，对精确定位心脏结构至关重要。
:::

## 🧠 深度精讲

*   **心脏的真实解剖方位**：视频纠正了一个常见的教学误区。在标准的解剖学姿势下，心脏并非像许多教科书插图（“情人节”位）那样心尖垂直向下。实际上，心脏位于胸腔的**中纵隔**，坐于**膈肌**之上，两肺之间。其**心尖**（较尖的一端）朝**前**、**向下**、并向**左侧**倾斜；而**心底**（大血管出入的较宽一端）则朝向**后方**。这种三维空间定位是理解心脏与周围结构关系的关键。

*   **“情人节”位 vs. 原位**：“情人节”位是为了便于在解剖台上观察和教学而采用的一种展示方式，类似于心脏被取出平放时的状态。而“原位”则指器官在活体内的自然位置和朝向。区分这两者对于将解剖知识应用于活体病人至关重要。

*   **心脏的“住所”：纵隔与心包**：心脏并非“悬浮”在胸腔中。它位于一个特定的解剖区域——**纵隔**（胸腔中部、两肺之间的区域）。具体来说，心脏占据**中纵隔**，这里也是心包、大血管根部及重要神经的所在地。心脏被一个称为**心包**的双层膜性囊包裹，心包起到固定、保护和润滑的作用。

## 📚 双语术语表 (Terminology)
| 英文术语 | 中文翻译 | 定义/解释 |
| :--- | :--- | :--- |
| Orientation | 方位/朝向 | 器官在空间中的位置和方向。 |
| Apex of the heart | 心尖 | 心脏左下部的尖端，朝前、下、左。 |
| Base of the heart | 心底 | 心脏上部较宽的部分，大血管出入处，朝后。 |
| In situ | 原位 | 指器官在生物体内自然的位置和状态。 |
| Thoracic cavity | 胸腔 | 由胸廓围成的体腔，容纳心肺等重要器官。 |
| Mediastinum | 纵隔 | 位于两侧胸膜腔之间、胸骨之后、脊柱之前的区域。 |
| Middle mediastinum | 中纵隔 | 纵隔的一部分，包含心脏、心包、大血管根部等。 |
| Diaphragm | 膈肌 | 分隔胸腔和腹腔的穹窿状肌肉，心脏坐于其上。 |
| Pericardium | 心包 | 包裹心脏和各大血管根部的纤维浆膜囊，分壁层和脏层。 |
| Valentine‘s position | “情人节”位 | 一种传统的心脏图示方式，心尖垂直向下，非自然解剖方位。 |
