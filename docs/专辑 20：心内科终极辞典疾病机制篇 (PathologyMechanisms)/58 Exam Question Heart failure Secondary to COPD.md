::: tip ⚡️ 核心考点 (30s速读)
*   **核心考点**：慢性阻塞性肺疾病（COPD）如何通过增加右心室后负荷，导致右心衰竭（肺源性心脏病）。
*   **临床意义**：理解心力衰竭的三种机制性病因（前负荷、后负荷、收缩力）是解题关键。COPD引起的肺动脉高压属于典型的“右心后负荷增加”型心衰。
:::

## 🧠 深度精讲

*   **心力衰竭的三大机制**：视频首先回顾了心力衰竭的三种基本病因。这是分析所有心衰问题的基石。
    1.  **前负荷原因**：指回流入心室的血量过多（舒张末期容积过大），如液体超负荷状态。
    2.  **后负荷原因**：指心室射血时需要对抗的阻力过高。左心后负荷增高见于体循环高血压、主动脉瓣狭窄；右心后负荷增高见于肺动脉高压。
    3.  **收缩力原因**：指心肌本身的泵血能力受损，如心肌梗死。

*   **COPD导致右心衰竭的病理生理**：这是本题的核心。
    1.  **起点**：患者患有COPD，肺部存在慢性炎症和通气不均。
    2.  **关键步骤**：为了优化通气/血流比，缺氧区域的肺血管会发生**收缩**，将血液重新分配到通气良好的区域。
    3.  **结果**：广泛的肺血管收缩导致**肺动脉压力升高**，即肺动脉高压。
    4.  **影响心脏**：肺动脉高压意味着右心室在向肺动脉射血时需要对抗更大的阻力，即**右心室后负荷增加**。
    5.  **最终结局**：长期的后负荷过重，导致右心室代偿性肥厚，最终失代偿，发生**右心衰竭**。这种由肺部疾病引起的心脏病，特称为**肺源性心脏病**。

*   **选项分析逻辑**：视频通过逐一排除错误选项，强化了上述逻辑。
    *   **左侧后负荷增加**：错误。COPD问题在肺部循环，不影响体循环（主动脉）压力。
    *   **右侧前负荷增加**：错误。COPD并未导致更多血液回流至右心。
    *   **右侧收缩力问题**：错误。题目未提示心肌本身受损（如心梗）。
    *   **左侧前负荷增加**：错误。肺部血流可能因血管收缩而减少，回流至左心的血量更可能减少而非增加。
    *   **双心室衰竭**：错误。其描述（右侧前负荷减少，左侧前负荷增加）与COPD的病理生理不符。
    *   **右侧后负荷增加**：**正确**。完美对应了COPD→肺动脉高压→右心后负荷增加→右心衰竭的链条。

## 📚 双语术语表 (Terminology)
| 英文术语 | 中文翻译 | 定义/解释 |
| :--- | :--- | :--- |
| Heart Failure | 心力衰竭 | 心脏泵血功能无法满足机体代谢需要的综合征。 |
| Pulmonary Hypertension | 肺动脉高压 | 肺动脉压力异常升高的病理状态。 |
| COPD | 慢性阻塞性肺疾病 | 一组以持续气流受限为特征的慢性肺部疾病。 |
| Afterload | 后负荷 | 心室开始收缩射血时需要克服的阻力。 |
| Preload | 前负荷 | 心室舒张末期容积，代表心肌收缩前的初长度。 |
| Contractility | 收缩力 | 心肌纤维本身收缩的能力，与前、后负荷无关。 |
| Cor Pulmonale | 肺源性心脏病 | 由肺部疾病或肺血管病变引起的右心室结构和/或功能改变。 |
| Systemic Hypertension | 体循环高血压 | 体动脉血压持续升高的状态。 |
| End-diastolic Volume | 舒张末期容积 | 心室在舒张末期充盈完成时的血容量。 |
