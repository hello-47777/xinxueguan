::: tip ⚡️ 核心考点 (30s速读)
*   **核心考点**：体循环动脉是将含氧血从左心室泵出，输送到全身组织（除肺以外）的血管。其核心功能是输送氧气和营养物质。
*   **临床意义**：动脉搏动是评估循环系统功能的重要指标，常用测量点（如桡动脉、颈总动脉）是临床体检和急救的关键部位。
:::

## 🧠 深度精讲

*   **体循环动脉的定义与功能**：体循环动脉是血管系统的一部分，特指将血液**从心脏运离**并送往全身各组织（体循环组织）的血管。在本视频讨论的成人循环中，这些血液是**含氧血**。其核心功能是作为“输送管道”，将左心室泵出的富含氧气和营养物质的血液，通过主动脉及其各级分支，最终送达体循环毛细血管网，与组织进行物质交换。

*   **体循环与肺循环、冠状循环的区别**：
    *   **体循环**：左心室 → 主动脉 → 全身动脉 → 毛细血管（物质交换）→ 全身静脉 → 右心房。运输含氧血。
    *   **肺循环**：右心室 → 肺动脉 → 肺毛细血管（气体交换）→ 肺静脉 → 左心房。肺动脉运输的是**脱氧血**，这是“动脉运输含氧血”这一通则的重要例外。
    *   **冠状循环**：是心脏自身的供血系统，属于体循环的一个特殊分支，但常被单独讨论。冠状动脉从升主动脉最先发出，为心肌供血。

*   **主要体循环动脉的起源与分布**：动脉的命名通常基于其所在的**身体区域**或**供应的主要器官**。
    1.  **升主动脉**：起自左心室主动脉瓣上方，向上走行。其最早的分支是**左、右冠状动脉**，供应心肌。
    2.  **主动脉弓**：呈弓形弯曲。发出三大分支，从右向左依次为：
        *   **头臂干（头臂动脉）**：向右上走行一短距离后，分为**右锁骨下动脉**（供应右上肢）和**右颈总动脉**（供应右头颈部）。
        *   **左颈总动脉**：直接发自主动脉弓，供应左头颈部。
        *   **左锁骨下动脉**：直接发自主动脉弓，供应左上肢。
    3.  **降主动脉**：主动脉弓之后的部分，分为胸主动脉和腹主动脉，沿途发出众多分支供应胸腔、腹腔及盆腔脏器（如肋间动脉、肾动脉、肠系膜动脉等）以及下肢（髂动脉等）。

*   **脉搏**：脉搏是动脉管壁随着心脏的收缩（射血）和舒张而产生的规律性搏动。在体表可触摸到搏动的动脉，即为**脉搏点**。
    *   **最常用的脉搏测量动脉**：
        *   **桡动脉**：位于手腕掌面外侧（拇指侧），是最常用的测量部位。
        *   **颈总动脉**：位于颈部气管两侧，常用于急救时快速评估循环状态。
        *   其他常用点还包括肱动脉（肘窝）、股动脉（腹股沟）和足背动脉等。

## 📚 双语术语表 (Terminology)
| 英文术语 | 中文翻译 | 定义/解释 |
| :--- | :--- | :--- |
| Systemic arteries | 体循环动脉 | 将含氧血从心脏输送到全身组织的血管。 |
| Aorta | 主动脉 | 体循环动脉的主干，起自左心室。 |
| Ascending aorta | 升主动脉 | 主动脉的起始段，向上走行，发出冠状动脉。 |
| Aortic arch | 主动脉弓 | 主动脉的弓形弯曲部分，发出供应头颈和上肢的主要分支。 |
| Brachiocephalic trunk | 头臂干 | 主动脉弓的第一个分支，供应右头颈部和右上肢。 |
| Pulse | 脉搏 | 动脉管壁随心脏搏动而产生的周期性起伏。 |
| Radial artery | 桡动脉 | 前臂的动脉之一，位于手腕部，是测量脉搏的常用部位。 |
| Common carotid artery | 颈总动脉 | 供应头颈部的主要动脉，位于颈部两侧。 |
| Coronary arteries | 冠状动脉 | 从升主动脉最先发出的分支，专为心脏心肌供血。 |
| Pulmonary arteries | 肺动脉 | 将脱氧血从右心室运往肺部的血管，是“动脉运输含氧血”规则的例外。 |
